# EXCEL与数据格式
## 1.假日期转化为真日期--WPS
选中假日期列 -> 数据-> 分列 -> 连续两个下一步 -> 选中日期YMD -> 确定
带小数点的先按照其格式用“分列大法”转化为对应的数据形式，在改单元格格式。

## 2.单元格格式--WPS
选中单元格，右键选择单元格格式/ctrl+1 快捷键。可以更改内容如下

数字：调整数值格式或者日期格式。<br>对齐：调整对齐方式。<br>字体：更改字体、字型、字号等。<br>边框：单元格边框设置。<br>填充：填充单元格背景颜色，标注数据经常使用。<br>保护：锁定单元格或者隐藏公式。

# 数据导入
## 1.CSV格式数据导入
新建空白工作簿 -> 数据 -> 获取数据 -> 导入数据 -> 文本格式（编码）以预览文件为准 -> 分隔符号一般选逗号，结合实际文件中的符号选择

## 2.txt格式数据导入
同上，最主要的是查看是否有中文乱码问题和分隔符号的选择

# EXCEL的快捷操作
## 1.文件相关
Ctrl + N 创建一个新的Excel工作簿<br>Ctrl + w 关闭当前工作簿<br>F12 将当前文件进行另存为<br>Ctrl + p 打印当前工作表
## 2.通用快捷键
Ctrl + Y 恢复撤销，可以用这个快捷键恢复被撤消的操作<br>Ctrl + F 弹出查找和替换对话框<br>Ctrl + G 显示定位对话框
## 3.表格选择
Ctrl + A <br>全选表格Ctrl + 上下左右 移动到数据区域的上下左Alt + Enter 在当前单元格内换行 (直接按 Enter会跳转到下一行)右边缘
## 4.单元格编辑
F2 编辑当前单元格（跟双击单元格效果一致）<br>Shift + Enter 上移一个单元格<br>Tab 右移一个单元格<br>Shift + Tab 左移一个单元格<br>Ctrl + Enter 输入完成，并停留在当前单元格<br>Ctrl + ;（分号） 输入当前日期（显示为年/月/日）<br>Ctrl + Shift + :  输入当前时间 (显示为小时/分钟)Ctrl + E 快速填充
## 5.EXCEL格式化
Ctrl + 1（数字1） 设置单元格格式<br>Ctrl + B 加粗或取消加粗<br>Ctrl + i 加斜体或取消斜体
## 6.Excel公式
F4 绝对引用和相对引用切换<br>Alt+= 自动插入求和公式<font color="red"> ！！！常用</font>
自动进行列差异比对：ctrl+\ 或者 选中内容ctrl+G 定位行差异或列差异<font color="red"> ！！！常用</font>
